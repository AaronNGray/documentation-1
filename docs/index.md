# What is Armbian? #

- Lightweight Debian and Ubuntu based distribution specialized for ARM developing boards, 
- It has powerfull build and software development tools,
- A vibrant community. 

# Which chips are supported?

- Allwinner A10, A20, A31, H3
- Amlogic S805
- Amlogic S905
- Actionsemi S500
- Freescale / NXP iMx6
- Marvell Armada A380
- Samsung Exynos 5422
